OS's ISO images To Use as VM's

To use a VM you will need to download the OS's ISO images. Here are where to find some of them:

  • Windows 10: https://www.microsoft.com/en-us/software-download/windows10
  • Windows 11: https://www.microsoft.com/software-download/windows11
  • Kali Linux: https://www.kali.org/get-kali/#kali-platforms
  • Linux Parrot: https://www.parrotsec.org/download/
  • Linux Tails: https://tails.net/install/
  • Linux Lite: https://www.linuxliteos.com/download.php

I recommend Linux Lite, because is a lightweight OS and friendly to non-Linux users. Also you can give a second life to your OLD hardware with this distro.
